# Instagram Reels 게시 체크리스트

## 게시 자료

- [ ] 최종 MP4 파일
- [ ] 커버 이미지
- [ ] `caption.md` 본문
- [ ] 해시태그
- [ ] CTA URL 또는 프로필 링크

## 계정 준비

- [ ] Instagram Professional 계정
- [ ] 연결된 Facebook Page
- [ ] Meta Business 설정
- [ ] 자동 게시가 필요하다면 Meta App과 Content Publishing 권한

## 공개 전 검토

- [ ] 실제 고객명, 이메일, 토큰, 비공개 파일 URL이 보이지 않는다.
- [ ] 화면 전환이 너무 빠르지 않고 주요 문구를 읽을 수 있다.
- [ ] Flowrit의 핵심 가치가 첫 3초 안에 드러난다.
- [ ] 커버 이미지에서 제품 화면이 명확하다.
- [ ] 배경 음악과 이미지 라이선스가 확인되었다.
