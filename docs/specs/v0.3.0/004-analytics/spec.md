## 배경 및 목적

현재 대시보드는 "진행 중 N건", "마감 임박 N건" 두 개의 숫자와 오늘 처리할 작업 목록만 제공한다. 사업자가 지난 한 달 동안 몇 건을 완료했는지, 수정 요청이 늘고 있는지, 팀원 간 작업량이 균등한지 파악할 방법이 없다.

엔터프라이즈 사용자(3인 이상 팀, 월 10건 이상 진행)는 데이터 기반으로 운영을 개선해야 한다. 이 스펙은 별도 "통계" 페이지를 추가하여 비즈니스 인사이트를 제공한다.

---

## 사용자 스토리

- **US-501**: 사업자로서, 이번 달에 완료한 프로젝트 수와 지난달 대비 추이를 확인하고 싶다.
- **US-502**: 사업자로서, 프로젝트당 평균 수정 요청 건수를 파악하여 작업 복잡도를 이해하고 싶다.
- **US-503**: 사업자로서, 팀원별로 현재 담당 중인 프로젝트 수와 미완료 수정 건수를 확인하고 싶다.
- **US-504**: 사업자로서, 월별 신규 의뢰 접수 추이를 확인하여 성수기·비수기를 파악하고 싶다.
- **US-505**: 사업자로서, 프로젝트에 예상 단가(예산)를 기록하고 월별 수익 현황을 확인하고 싶다.

## 기능 요구사항

### 통계 페이지 추가

- **FR-501**: 사이드바에 "통계" 메뉴(`/analytics`)를 추가한다. 기간 필터(이번 달 / 지난 3개월 / 지난 6개월 / 올해)를 제공한다.

### 프로젝트 완료 현황

- **FR-502**: 선택 기간 내 완료된 프로젝트 수를 카드로 표시한다. "완료" 기준은 `isProjectDone()` 로직(마지막 단계 도달 또는 `completedAt` 있음)을 따른다.
- **FR-503**: 월별 프로젝트 완료 수를 바 차트로 표시한다. 최근 6개월을 기본으로 한다.
- **FR-504**: 선택 기간 내 신규 생성된 프로젝트 수와 완료된 프로젝트 수를 함께 표시하여 미처리 누적 추세를 파악할 수 있도록 한다.

### 수정 요청 분석

- **FR-505**: 선택 기간 내 접수된 수정 요청 총 건수를 표시한다.
- **FR-506**: 수정 요청 출처(사업자 직접 등록 `MANUAL` vs. 고객 포털 `PORTAL`) 비율을 원형 차트로 표시한다.
- **FR-507**: 프로젝트당 평균 수정 요청 건수를 표시한다 (완료된 프로젝트 기준).

### 팀 워크로드

- **FR-508**: 현재 시점 기준으로 팀원별 담당 중인 진행 중 프로젝트 수와 미완료 수정 요청 수를 표로 표시한다.
- **FR-509**: "담당자 미지정" 프로젝트·수정 요청도 별도 행으로 표시한다.

### 의뢰 접수 추이

- **FR-510**: 월별 신규 의뢰(`Inquiry`) 접수 수를 선 그래프로 표시한다. 최근 6개월을 기본으로 한다.

### 프로젝트 단가(예산) 기록

- **FR-511**: `Project` 모델에 `budget Int?` 필드를 추가한다. 단위는 원(KRW)이며 선택 입력이다.
- **FR-512**: 프로젝트 생성 폼과 프로젝트 상세 헤더에 예산 입력/편집 필드를 추가한다.
- **FR-513**: 통계 페이지에서 선택 기간 내 완료된 프로젝트들의 예산 합계("예상 수익")를 카드로 표시한다. 예산 미입력 프로젝트는 합산에서 제외하고 제외 건수를 안내한다.

## 비기능 요구사항

- **NFR-501**: 통계 페이지의 데이터 조회는 Server Component에서 수행하며, 초기 로딩 시간은 3초 이내를 목표로 한다.
- **NFR-502**: 차트 라이브러리는 `recharts`를 사용한다. shadcn/ui의 Chart 컴포넌트(`components/ui/chart.tsx`)를 통해 래핑한다.
- **NFR-503**: 통계 데이터는 워크스페이스 범위 내로 엄격히 제한된다. 다른 워크스페이스 데이터를 조회할 수 없다.

---

## 수용 기준

- **SC-501** (`FR-501`): 사이드바에 "통계" 링크가 표시되며, 클릭 시 `/analytics` 페이지로 이동한다.
- **SC-502** (`FR-502`, `FR-503`): 통계 페이지에서 "이번 달" 필터 선택 시 현재 달의 완료 프로젝트 수가 표시된다. 바 차트에는 최근 6개월 데이터가 시각화된다.
- **SC-503** (`FR-505`, `FR-506`): 수정 요청 출처 원형 차트에서 MANUAL·PORTAL 비율이 시각화된다.
- **SC-504** (`FR-508`): 팀 워크로드 표에서 현재 각 팀원의 담당 프로젝트 수와 미완료 수정 건수가 표시된다.
- **SC-505** (`FR-511`, `FR-512`): 프로젝트 생성 폼에서 예산을 입력하면 저장되고, 상세 헤더에서 수정할 수 있다.
- **SC-506** (`FR-513`): 통계 페이지에서 선택 기간 내 완료 프로젝트들의 예산 합계가 "예상 수익" 카드에 표시된다.
